# Project Brief: nirs

**Project Name:** nirs

**Description:** A Rust implementation of the `ni` command-line tool, providing a unified interface for managing packages across different package managers (npm, yarn, pnpm, bun, etc.).

**Core Requirements:**

- Automatically detect the package manager used in a project.
- Provide a simplified command-line interface for common package management operations (install, run, upgrade, uninstall, etc.).
- Support multiple package managers: npm, yarn, pnpm, and bun.
- Offer logging with different verbosity levels.
- Provide a way to install the tool and its associated binaries easily.

**Goals:**

- Simplify package management for developers.
- Eliminate the need to remember package manager-specific commands.
- Provide a consistent experience across different projects.
- Improve developer workflow efficiency.
